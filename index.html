Interfaces Web

Sobre a disciplina e tecnologias abordadas.

Na Unidade Curricular de Desenvolvimento de Interfaces Web abordamos HTML, CSS, JavaScript, React e Next.js. Focamos boas práticas de semântica, acessibilidade, responsividade e padrões de UI modernos.

  • Estruturar páginas semânticas (HTML5)
  • Estilizar interfaces (CSS/Flex/Grid)
  • Tornar páginas interativas (JavaScript)
  • Componentização (React/Next.js)
